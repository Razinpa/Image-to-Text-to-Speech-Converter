HOW IT WORKS?

What is Image to text to Speech convertor?

Image,text to speech is a Webapplication for Image proccesing using Python and flask. The project is about recognizing and extracting the texts from any Image and translating that text into different language and thus creating an audio file for that text. You can listen to any image containing text. Our product runs on the best text-to-speech engine technology which guarantees human-like voice, smart text recognition and natural sounding

How to Convert the Image to Text?

For the Extraction of text from an image, Press the button "Image to Text" after that you have to upload the Image from the file. Then the text will be extracted from the image and from there you can Translate that text to different languages and get an audio file of it.

How is the Working of the WebApplication?

Using pytesseract images will be read and recognized, pytesseract is an Optical Character Recognition tool for python.Using Googletrans the extracted text will be translated into different languages and then the Google Text to Speech API, commonly known as the gTTS API is used to create the audio file of it.

How to Convert the Text to Voice?

You can also Translate and Convert the text itself without uploading an Image, for that you have to press the button"Text to Voice" and there you can type the text and get the translation and the audio file of the text.
